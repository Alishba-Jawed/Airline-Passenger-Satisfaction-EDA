# ✈️ Airline Passenger Satisfaction: Data Cleaning & Exploratory Data Analysis

## 📌 Project Overview

This project focuses on cleaning and analyzing the Airline Passenger Satisfaction dataset using Python. The goal is to preprocess the data, handle missing values, remove duplicate records, and perform Exploratory Data Analysis (EDA) to better understand passenger satisfaction.

---

## 🎯 Objectives

- Explore the dataset
- Clean and preprocess the data
- Handle missing values
- Remove duplicate records
- Perform Exploratory Data Analysis (EDA)
- Identify patterns related to passenger satisfaction

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📊 Project Workflow

1. Import libraries
2. Load dataset
3. Explore dataset
4. Data cleaning
5. Handle missing values
6. Remove duplicates
7. Exploratory Data Analysis
8. Draw conclusions

---

## 📈 Key Insights

- Missing values were successfully handled.
- Duplicate records were removed.
- The dataset was cleaned and prepared for analysis.
- Visualizations revealed useful patterns in passenger satisfaction.
